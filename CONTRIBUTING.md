# Contributing to MovieVerse

First off, thank you for considering contributing to MovieVerse! 

## Code of Conduct
By participating in this project, you are expected to uphold standard professional conduct. Be respectful, helpful, and collaborative.

## Development Workflow

1. **Fork the repository** and clone it locally.
2. **Create a new branch** for your feature or bugfix following this naming convention:
   - `feat/your-feature-name`
   - `fix/issue-description`
   - `docs/update-readme`
3. **Commit your changes** using [Conventional Commits](https://www.conventionalcommits.org/):
   - `feat: add robust watchlist functionality`
   - `fix: resolve hydration error on movie detail page`
4. **Push to your fork** and submit a Pull Request against the `main` branch.

## Architectural Guidelines
Before submitting code, please ensure it adheres to the rules defined in our master playbook (`AI_ENGINEERING_PLAYBOOK.md`), specifically:
- No Glassmorphism or heavy shadows.
- Strict adherence to SOLID principles.
- Use of TanStack Query for server state.
