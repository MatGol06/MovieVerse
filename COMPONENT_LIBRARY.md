# Component Library Inventory

## 1. Buttons
- **Primary Button:** Background `#2563EB`, Text `#FFFFFF`. Used for primary actions (e.g., "Add to Watchlist").
- **Secondary Button:** Background `#333333`, Text `#FFFFFF`. Used for alternative actions (e.g., "Remove").
- **Ghost/Text Button:** Transparent background, text `#B3B3B3`, hover background `#1E1E1E`.

## 2. Inputs
- **Search Input:** Background `#1E1E1E`, Border `#333333`. Placeholder text `#B3B3B3`. Must have clear focus state.

## 3. Cards
- **Movie Card:**
  - Background: `#1E1E1E`
  - Border: 1px solid `#333333`
  - Radius: `8px`
  - Content: Poster Image (top), Title (truncate 1 line), Release Year, Rating.
  - Hover State: Border changes to `#B3B3B3`, cursor pointer. (NO massive scale up or shadow increase).

## 4. Feedback
- **Loading Skeleton:** Solid `#333333` blocks with subtle pulse animation.
- **Toast Notification:** Background `#1E1E1E`, Border `#333333`, positioned bottom-right.
