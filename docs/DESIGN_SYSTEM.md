# Design System (Design Tokens)

Strict adherence to a pragmatic, accessible, SaaS-grade design system.

## Color Palette (Accessible Dark Theme)
- **Background (Base):** `#121212`
- **Surface (Cards/Modals):** `#1E1E1E`
- **Primary Action (Brand):** `#2563EB` (Accessible Blue)
- **Borders / Dividers:** `#333333`
- **Text (Primary):** `#FFFFFF`
- **Text (Secondary):** `#B3B3B3`
- **Error / Destructive:** `#CF6679`

## Typography (Inter or Roboto)
- **Font Scale:**
  - H1: 32px (Bold, tight tracking)
  - H2: 24px (Semi-Bold)
  - H3: 20px (Medium)
  - Body (Primary): 16px (Regular, optimal line-height 1.5)
  - Body (Small): 14px (Regular)

## Spacing System
Base unit: `4px`
- `xs`: 4px
- `sm`: 8px
- `md`: 16px
- `lg`: 24px
- `xl`: 32px
- `2xl`: 48px

## Border Radius
- **Inputs/Buttons:** `4px` (`rounded`) - Professional, slightly softened edges.
- **Cards/Modals:** `8px` (`rounded-lg`) - Clear containment without excessive roundness.
